# Website Komunitas Pushbike Jakarta (PBJ) 🚲

[cite_start]Website resmi untuk **Pushbike Jakarta (PBJ)**, sebuah komunitas non-profit yang menjadi wadah berkumpulnya anak-anak penggemar olahraga sepeda tanpa pedal (balance bike) di Jakarta[cite: 1, 2, 3, 5].

## 📝 Deskripsi Proyek
[cite_start]Pushbike atau *balance bike* adalah sepeda roda dua tanpa pedal, rantai, atau rem yang dirancang khusus untuk anak usia 18 bulan hingga 7 tahun[cite: 4, 6]. [cite_start]PBJ berkomitmen untuk membina dan meningkatkan prestasi anak-anak usia 2-7 tahun dalam beraktivitas maupun berkompetisi[cite: 6].

- **Target Market:** Mama dan Papa Racing.
- **Teknologi Backend:** Python Django.
- **Database:** PostgreSQL.
- **Style Desain:** Antigravity (Modern, Clean, Floating Elements).
- [cite_start]**Tahun Berdiri:** 2018 - sekarang[cite: 6].

* [cite_start]**Visi:** Kita bisa bersatu, kita jadi satu, kita menjadi besar dan dikenal semua kalangan pecinta Pushbike[cite: 6].
